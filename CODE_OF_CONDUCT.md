# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## TL;DR

- Be kind. Disagree on technique, not on people.
- No harassment, no discrimination, no doxing, no spam.
- We're a volunteer project. Patience cuts both ways.

## Reporting

Email Wesley Simplicio at the address listed on the GitHub profile of `@wesleysimplicio`. Reports stay private. We'll respond within a reasonable window — usually a few days.

## Enforcement

Maintainers may warn, temporarily restrict, or permanently ban contributors whose behavior violates this code. Decisions are documented privately in the maintainer log.

## Scope

This applies to all project spaces — issues, PRs, discussions, the Discord channel if/when one exists, and any in-person events held under the project name.
